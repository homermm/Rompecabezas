# Rompecabezas

Juego de rompecabezas online con piezas que encajan entre sí. Arrastra las piezas para armar la imagen.

## Demo

[Ver demo en vivo](https://tu-url-aqui.com)

## Características

- Piezas con bordes redondeados que encajan entre sí
- Tres niveles de dificultad (24, 54 y 96 piezas)
- Las piezas se agrupan automáticamente al conectarse
- Guarda el progreso automáticamente en el navegador
- Imagen guía opcional
- Temporizador
- Pantalla completa

## Instalación

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Los archivos de producción quedan en la carpeta `dist/`.

## Stack

- TypeScript
- Vite
- Canvas API
